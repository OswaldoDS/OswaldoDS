### 📂 Portafolio de Evidencias (Proyectos Clave)

Estos son los proyectos donde he aplicado conceptos avanzados de ingeniería:

#### Sistema Gestor de Usuarios - Spring Boot MVC 
* **Descripción:** Este proyecto es una aplicación web robusta diseñada para la gestión y mantenimiento de usuarios (CRUD). Implementa una interfaz de usuario moderna y responsiva utilizando Bootstrap y sigue los estándares de la arquitectura Model-View-Controller (MVC) para garantizar un código organizado, escalable y fácil de mantener.
* **Reto técnico:** Implementación de persistencia de datos eficiente mediante MySQL a través de JDBC, garantizando la integridad de la información y la optimización de consultas SQL complejas.
* **Link:** https://github.com/OswaldoDS/Spring-boot-MVC

#### E-commerce Tinta Nómada
* **Descripción:**Digitalización integral de un modelo de negocio de sublimación y personalización bajo metodologías ágiles.
* **Reto técnico:** Implementación de SCRUM con rotación de roles para optimizar el ciclo de desarrollo y digitalizar el flujo de atención al cliente, logrando una gestión de pedidos 100% eficiente.
* **Link:** https://github.com/jesusmoralesgarita/Turing-s-Bug-Hunters

#### Sistema IoT de Monitoreo y Control - Optimización de Cultivos
* **Descripción:** Desarrollo de un algoritmo de control para variables ambientales que optimizó el crecimiento de producción de setas rosas en un 18.6%, integrando sensores con las APIs de Google Sheets y Telegram.
* **Reto técnico:** Optimización de la producción en un 18.6% y reducción del tiempo de respuesta del sistema de 35 a 30 días mediante el diseño de algoritmos de control y la integración de las APIs de Google Sheets y Telegram.
* **Link:** https://github.com/OswaldoDS/Proyecto-de-titulacion
